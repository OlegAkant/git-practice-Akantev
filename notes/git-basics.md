Что такое Git?
Git — это самая популярная распределенная система управления версиямисозданная Линусом Торвальдсом в 2005 году. Она позволяет отслеживать изменения в файлахработать над проектами параллельносохранять историю версий и легко откатываться к предыдущим состояниям.

Что такое репозиторий?
Репозиторий Git — это цифровое хранилище (папка) вашего проекта, которое отслеживает все изменения файлов, историю их версий и служебную информацию.

Зачем нужны коммиты?
Коммиты (commit) в Git — это «снимки» состояния проекта, которые фиксируют изменения файлов в определенный момент времени. Они необходимы для сохранения истории разработки, возможности безопасного отката к предыдущим рабочим версиям, совместной работы над кодом и контроля качества, позволяя разбивать работу на этапы.
